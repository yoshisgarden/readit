# 変更履歴

## v1.1.1
- 辞書の検索ボックスに「クリア（×）」ボタンを追加（タップで入力を消去＋キーボードを閉じる）
- 一部端末（AQUOS sense7 など）で、検索時にソフトキーボードが下部ナビ（ホーム等）を覆ってタップできない問題に対応
  - キーボード表示中も下部ナビをキーボードの上に表示（imePadding）
  - `adjustResize` を明示し、機種差によるキーボード挙動を安定化

## v1.1.0
- フレーズ辞書を **75 → 945 件**に大幅拡充（7カテゴリ）
  - API Docs 201 / Error Messages 150 / CLI・Terminal 150 / Abbreviations 100 / PR Review 118 / Claude Code 107 / GitHub 119
- 辞書カテゴリの並び順を見直し（API Docs を先頭に）
- カテゴリに **CLI / Terminal**（git の出力含む）と **Abbreviations（略語・頭字語）** を追加
- 単語帳：次のカードに進む際に答えが一瞬見えてしまう不具合を修正
- 練習テスト：回答後に「正解／不正解」を表示
- アプリ更新でフレーズ件数が変わると自動で再取り込み（お気に入りは英語テキストで引き継ぎ）
- アプリ内の開発支援を Stripe に統一

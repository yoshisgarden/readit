# 変更履歴

## v1.1.12
- 進捗に「今日の目標」カードを追加：今日の学習時間／目標分の達成バー（達成で🌸、未達は「あと○分」）と、直近の目標達成日数を表示
- 単語帳のカードにも意味の吹き出しを追加：カタカナ語など「用語（意味）」の単語は、用語を大きく表示し、その下の吹き出しに意味を表示（表裏どちらの面でも）

## v1.1.11
- 難読漢字を読みやすく：「冪等」→「べき等」に変更
- 不透明なカタカナ専門用語にさらに意味を付与（ラッパー／デシリアライズ／ロケール／ハンドシェイク／ゲートウェイ など約30語）

## v1.1.10
- 【不具合修正】一部端末（Pixel 8 など）でホーム等のタップが効かないことがある問題を修正。意味の吹き出しを、外側タップを横取りしないインライン表示に変更
- 純カタカナ語に意味データを一括付与（プルリクエスト／デフォルト値／必須パラメータ／リンカエラー など約50語）。タップで意味が出る単語を拡大
- 穴埋めクイズ：日本語訳のヒントに答えの語（GUI など）がそのまま出ないようマスク

## v1.1.9
- カタカナ訳に意味が併記された単語は、普段は用語だけを表示（背景色でタップ可能と分かるように）。タップすると吹き出しで意味を表示し、再タップ／外側タップで閉じる（辞書の一覧・詳細）

## v1.1.8
- 進捗グラフを刷新：縦軸の目盛りライン（分）と赤い目標ラインを表示。棒は固定幅で1日分でも見やすく
- 設定に「学習目標（1日の学習時間：5 / 10 / 15 / 30 分）」を追加。グラフの目標ラインに反映
- 辞書検索の並び順を改善：部分一致（LIKE）はそのままに、先頭一致を上位に表示

## v1.1.7
- 「エラー解析」を「エラー・コマンド解析」に変更し、エラーとコマンド（CLI / Terminal）から出題（コマンドがエラー扱いで出る不具合を解消）
- クイズの選択肢を同じカテゴリから選ぶようにし、明らかに無関係な選択肢を排除
- 穴埋めで文頭に入る場合は、選択肢の頭文字を大文字に統一
- モード名・問いの文の表現を調整
- 単語帳：「用語（意味）」のような表記を、読みやすいよう「（」の前で改行
- 単語帳：めくったときにカードの位置がずれないよう、下の領域を固定高に調整
- Claude Code の進行中ワード（Baking など 45 語）の和訳を、直訳をやめて自然な日本語に全面見直し
- 進捗グラフ：1日分でも見やすいよう棒を固定幅にし、各日の学習時間（分）を棒の上に表示。上限値（最大値に合わせた自動スケール）も表示
- 「please rebase（リベースしてください）」に意味の補足を追記
- ホームの「今日の復習: 0枚」を「今日の学習: N枚」に変更（復習期限カード＋未学習の新規カードを合算し、上限20）。新規ユーザーでも0にならず分かりやすく。完了時は「今日のカードは完了！」を表示
